# Governance Evidence Ledger

The registry authority records governance events in a tamper-evident evidence ledger.

Evidence events may include:

- governance check updates
- waiver issuance
- waiver expiry
- revocations
- readiness-impacting state changes

## Evidence Principles

- append-only event recording
- cryptographic chaining
- timestamped governance history
- replayable governance state references

This evidence model supports auditability and trust in governance decisions.
